# steves: Rick Steves' Europe Episodes for Teaching Data Analysis

Tidy snapshot of every episode of the public-television travel series
'Rick Steves' Europe' (2000-2025), enriched with IMDB ratings, geocoded
destinations, ISO country codes, episode thumbnails, and descriptive
summaries. Designed as a companion dataset for introductory data
analysis and visualization in the spirit of the 'moderndive' textbook:
every row is an episode, every column is a candidate for a plot or a
join.
